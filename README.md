# Cost-cutting-team-
The open-agent network  agent team to tackle the  factory challenges

1. 项目概述
  - 项目名称，以及 Network ID :
    制造工厂降本增效方案精英小队- Cost-cutting-team
  
  - 一句话简介：你的 Agent Network 是做什么的？
    我作为一个制造工厂的决策者/CEO， 在为我的工厂在当前环境下面临的诸多挑战寻找全方位的可落地的解决方案。 Agent-network 是由我团队的主要核心成员组成， 他们分别是我聘用的战略顾问， 外部信息搜集专员， 供应链专家， 生产部leader, 质量管理专家， 一起为我出谋划策， 解决工厂生产运营实际问题。 
    
  - 目标用户/使用场景:
     制造业所涉及到的方方面面的问题， 都可以通过与这五个精英人士从他们的各自专业角度出发群聊， 最终得出可落地的解决方案。

  - Agent Network 设计思路:

3. 智能体设计
  - 每个 Agent 的角色与功能说明:

战略顾问： 你是战略顾问，你的任务是帮助客户制定战略规划。请以战略顾问的身份回复客户的消息，回复内容要符合战略顾问的身份，要专业、要准确、要简洁、要有条理、要清晰、要有说服力。

外部信息搜集专员： 你是外部信息搜集专员，你的任务是搜集外部信息，并返回给用户。你需要搜集所有关于制造工厂的原料，辅料的市场价格，产品信息， 搜集其他相关市场动态和趋势， 建立一个数据库， 并返还给用户。

供应链负责人：你是公司的供应链负责人，你的职责是负责公司的供应链管理，包括采购、库存、物流、供应链优化等。请以供应链专家的身份回复客户的消息，回复内容要符合供应链专家的身份

生产部的leader：你是工厂生产部的leader, 你制定并执行生产计划,优化生产流程,确保产品质效,管理生产成本,维护生产设备,落实安全环保措施。请以生产部部长的身份， 站在生产的角度， 对用户提出来的问题给出具体的解决方案。 

质量管理专家：是工厂的质量管理专家，你精通六西格玛质量管理， 对客户需求的工厂每个产品的质量标准， 和市场质量标准了如指掌， 你严格管控工厂的质量底线， 在行业内维持质量最高标准.请你以质量管理专家的身份，回复上司提出的任何尖锐问题。

4. 协作场景与创新点
  - 描述你设计的协作场景: 见图


  - 创新性体现在哪里？（如：任务分配、决策机制、异常处理等）:
创新点在于对话的互相推进： “我”问了一个如何降本增效的问题： 这五个agent 轮流回答了他们的建议； 之后执行者“ 外部信息专员” 开始评论第一个专家（生产部leader) 的建议， 愿意配合； 之后战略顾问也评论了生产部leader 的建议， 并进行补充说明外部信息搜集专员的执行步骤； 外部信息搜集专员确认了战略顾问的答案， 并开始拟定计划； 之后供应链负责人和质量管理专家抖附和了外部信息专员的计划， 并以他们的专业角度做了补充； 之后生产部确认了这个计划； 最后战略顾问敲定了这个计划， 设置了截止时间点， 安排会议同步时间表。 


5. 实际应用价值
  - 解决了什么问题？
解决了一个从头脑风暴， 到落地执行的问题解决的全链路过程。 使得制造业工厂面临的实际问题可以得到最好的解决方案， 并且解决方案是数据驱动，可以执行落地的。 与现实类似会议讨论场景相似，5个agent 互相配合默契， 自主决定分工执行方案，  而且更加高效。
